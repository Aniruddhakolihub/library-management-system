# 📚 Library Management System

## 🚀 Overview
This is a full-stack Library Management System developed using **Java Spring Boot** and **MySQL**. It helps manage books, students, and issue/return operations efficiently.

---

## 🛠️ Tech Stack
- Backend: Java, Spring Boot
- Database: MySQL, Hibernate (JPA)
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Tools: Maven, Git

---

## ✨ Features
- 📖 Add, update, delete books
- 👨‍🎓 Manage student records
- 🔄 Issue and return books
- 📊 Track issued books and history
- 🔐 Admin dashboard
